# Game Demo using the SFML and ZeroMQ frameworks

## For macOS

1. Install SFML, ZeroMQ and CMake, if needed
2. After cloning this repo, build it with
   ```
   cmake -B build
   cmake --build build
   ```
3. Launch it like this
    ```
    build/bin/GameDemo

    ```
4. Enjoy!
